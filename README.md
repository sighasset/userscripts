# Userscripts

A collection of userscripts and userstyles for various websites

## Legend

- Userscript 📜
- Userstyle 🎨

## How to Use

### Userscripts 📜

1. Install a userscript manager, such as Tampermonkey or Violentmonkey
2. Open the README for the desired website
3. Click the **Install** link for the userscript
4. Confirm the installation in the userscript manager of your choice

### Userstyles 🎨

1. Install a userstyle manager, such as Stylus
2. Open the README for the desired website
3. Click the **Install** link for the userstyle and confirm the installation
4. Confirm the installation in the userstyle manager of your choice
