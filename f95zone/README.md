# F95zone

A collection of userscripts and userstyles for **[F95zone](https://f95zone.to/)**

Navigate pages with **F1** (previous) and **F2** (next).

### Links

- 📥 **Install:** **[page-nav.user.js](https://raw.githubusercontent.com/sighasset/userscripts/refs/heads/main/f95zone/page-nav/page-nav.user.js)**

### Links
